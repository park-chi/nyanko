# てのりニャンコ
ニャンコをてにのせるアプリ<br>

**【ニャンコをてにのせる方法】**<br>
[Build](https://github.com/park-chi/nyanko/tree/release/1.0.00/Build)フォルダをダウンロード<br>
→nyanko.exeを起動<br>
→Webカメラの映像がアプリに取り込まれ、画面に表示されます<br>
→手をカメラに見せると、手の上に猫が表示され、手を追従します<br>
<br>
**v1.0.00**<br>
・手のひらの上に猫を表示する機能を実装<br>
・スクリーンショット撮影機能を実装<br>
-----------------------------<br>
**今後の拡張展望**<br>
・猫に触れた際のアクション・音を追加<br>
・手の大きさに合わせて表示される猫の大きさを変更<br>
・複数種類の猫に対応<br>
・カメラが繋がれていない際の警告表示<br>
